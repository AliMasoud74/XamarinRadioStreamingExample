# XamarinRadioStreamingExample
## Setup
On _/iOS_
Set your streaming domain in _StreamingService.cs_. Change `dataSource`.
Allow transport in _Info.plist_ secure domains (change your.domain.com).

On Android
Set your streaming domain in _StreamingService.cs_. Change `dataSource`.
